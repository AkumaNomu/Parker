# Development

## Prerequisites

- Windows 10 or 11.
- Stable Rust with `rustfmt` and `clippy`.
- FFmpeg on `PATH`, beside `parker.exe`, or configured with `PARKER_FFMPEG`.
- Tesseract on `PATH`, in a standard installation directory, beside the
  executable, or configured with `PARKER_TESSERACT`.

Cargo downloads the `image` and `rqrr` crates during the first build. The lock
file is generated by Cargo and should be committed after the first successful
Windows build.

## Checks

```powershell
cargo fmt --all -- --check
cargo clippy --all-targets
cargo test
cargo build --release
```

Parker is Windows-only. Run checks on Windows or with a complete Windows target
toolchain.

## Manual test matrix

1. Select a QR URL and verify it is copied and opened once.
2. Set `PARKER_QR_AUTO_OPEN=0` and verify the URL is only copied.
3. Select multiple QR codes and verify all payloads are copied line-by-line.
4. Select source code in light and dark themes; verify code classification and
   indentation retention.
5. Select a two-column and multi-column table; paste into Excel and verify cell
   separation.
6. Force `PARKER_OCR_MODE` to each supported value and verify behavior.
7. Record regions on the primary monitor and monitors arranged left or above it.
8. Move the pointer through the region and confirm the cursor never appears.
9. Stop a short and a long recording; verify the temporary MKV is removed and
   the MP4 is playable and copied as a file.
10. Set different CRF, preset, and frame-rate values and inspect output.
11. Cancel both selectors with `Esc` and right-click.
12. Verify toast feedback without stealing keyboard focus.
13. Remove FFmpeg and verify smart capture still works.
14. Remove Tesseract and verify QR detection and recording still work.
15. Set `PARKER_KEEP_OCR_CAPTURE=1` and verify BMP retention.
16. Force a post-process failure and verify the `.capture.mkv` is preserved.
17. Exit while recording and verify the capture is finalized and copied.

## Versioning

Update `Cargo.toml` and `CHANGELOG.md`, regenerate and commit `Cargo.lock`, then
push a `vX.Y.Z` tag to create a GitHub release package.
